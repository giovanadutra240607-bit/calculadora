
## Funcionalidades

- Soma, subtração, multiplicação e divisão
- Potência e raiz quadrada
- Porcentagem
- Média de dois números
- Verificação de divisão por zero
- Verificação de raiz quadrada negativa

## Organização do projeto

```text
.
├── Main.java    # Código da calculadora
├── README.md    # Explicação do projeto
└── .gitignore   # Arquivos ignorados pelo Git
```

## Conceitos usados

- `Scanner`: recebe os dados digitados pelo usuário.
- `while`: mantém a calculadora aberta até o usuário escolher sair.
- `switch`: executa a operação escolhida.
- `if` e `else`: verificam situações como divisão por zero.
- `double`: permite trabalhar com números decimais.
- `Math.pow()`: calcula uma potência.
- `Math.sqrt()`: calcula uma raiz quadrada.

## Como executar

É necessário ter o Java JDK instalado.

No terminal, entre na pasta do projeto e compile:

```bash
javac Main.java
```

Execute a calculadora:

```bash
java Main
```

## Exemplo

```text
Digite a opção: 1
Digite o primeiro número: 10
Digite o segundo número: 5
Resultado: 15.0
```
