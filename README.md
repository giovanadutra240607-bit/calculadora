# Calculadora em Java

Calculadora desenvolvida em Java para execução via terminal, permitindo realizar operações matemáticas básicas e avançadas por meio de um menu interativo.

## Funcionalidades

* Soma
* Subtração
* Multiplicação
* Divisão
* Potência
* Raiz quadrada
* Cálculo de porcentagem
* Média de dois números
* Validação de divisão por zero
* Validação de raiz quadrada de números negativos

## Tecnologias Utilizadas

* Java
* Scanner
* Estruturas de repetição (while)
* Estruturas de decisão (if e switch)
* Classe Math

## Objetivo

Projeto desenvolvido para praticar lógica de programação, entrada e saída de dados, estruturas de controle e operações matemáticas utilizando Java.
